Overview

This unit contains a set of three speed button derived components that either display pop-up menus associated with the buttons or form connected button / menu button pairs. The components are:

  • TPJMenuSpeedButton – can display an associated pop-up menu when clicked.
  • TPJLinkedMenuSpeedButton – links to a master TPJLinkedSpeedButton button and displays an associated pop-up menu when clicked.
  • TPJLinkedSpeedButton – a speed button that can link to and control a slave TPJLinkedMenuSpeedButton that displays a pop-up menu.
Development of the menu speed buttons is now frozen
These components were originally written back in 2001 to get round a well known display problem that Delphi's tool button control had with some current display drivers (as on an old Dell machine of mine). I used speed buttons to avoid having to use tool buttons, and needed a button that supported drop down menus, so wrote the components.
Now that those driver problems are long gone, the original justification has disappeared: these days I use standard tool buttons and action bars.
Furthermore, the design of the menu speed button components doesn't lend itself easily to the application of XP themes, let alone Vista look and feel, so I'm not going to add this feature.
So, the final release 1.1 was issued on 15th August 2007 and it added a feature requested by a user. At the same time I've licensed the code under the Mozilla Public License. I'll be leaving the code on the site for the foreseeable future, but will not be updating it.
If you want to take on maintenance please go ahead. The download includes the components' source files and other assets, and all source files and bitmaps required to build (and change) the help file. I'd be interested to know about any changes you make the the components.

Installation

The Menu Speed Button components, documentation and demo program are supplied in a zip file. Before installing you need to extract all the files from the zip file, preserving the directory structure. The following files will be extracted:

  • PJMenuSpeedButtons.pas – component source file.
  • PJMenuSpeedButtons.dcr – resource file containing component palette glyphs.
  • PJMenuSpeedButtons.res – resource file containing default glyph for TPJLinkedMenuSpeedButton component.
  • PJMenuSpeedButtons.hlp – help file that integrates with the Delphi 3 to 7 OpenHelp system.
  • PJMenuSpeedButtons.als – a-link keyword file for integration with the Delphi 6 or 7 OpenHelp system.
  • PJMenuSpeedButtons.rc – source code for PJMenuSpeedButtons.res.
  • DropDown.bmp – default gylph referenced by PJMenuSpeedButtons.rc.
  • ReadMe.htm – this file.
  • ChangeLog.txt – component's change log.
  • Demo\DelphiDabbler.ico – demo program icon.
  • Demo\PJMenuSpeedButtonsDemo.dpr – demo project file.
  • Demo\PJMenuSpeedButtonsDemo.res – icon resource for demo.
  • Demo\PJMenuSpeedButtonsDemo.cfg – config file for compiling demo project.
  • Demo\FmPJMenuSpeedButtonsDemo.pas – demo form source file.
  • Demo\FmPJMenuSpeedButtonsDemo.dfm – demo form file.
  • Help\PJMenuSpeedButtons.hpj – help project file.
  • Help\PJMenuSpeedButtons.rtf – source document used in compiling help file.
  • Help\PJLinkedMenuSpeedButton.bmp – bitmap used in compiling help file.
  • Help\PJLinkedSpeedButton.bmp – bitmap used in compiling help file.
  • Help\PJMenuSpeedButton.bmp – bitmap used in compiling help file.

You can now proceed to install the components using the emphasised files above. Installation methods vary between Delphi versions, so follow the instructions that apply to your version. Note that $(DELPHI) represents the path where the appropriate version of Delphi was installed. Similarly $(BDS) refers to the installation path of a version of the Borland Development System.

  • Installation with Delphi 2005 & 2006
  • Installation with Delphi 4 to 7

Installation with Delphi 2005 & 2006

The components must be installed into a Delphi Win32 design package. It is assumed that you will be installing into an existing package. You may decide to install into the "user" package: $(BDS)\Lib\dclusr.dpk.

  1. Decide upon a suitable location into which to install the components. This may be a subdirectory of your Borland Studio Projects or $(BDS)\lib folders.
  2. Copy PJMenuSpeedButtons.pas, PJMenuSpeedButtons.dcr and PJMenuSpeedButtons.res to the chosen installation folder.
  3. Start the Development System (selecting the Delphi Win32 personality if appropriate) then:
    • Open the required package. Select File | Open menu and choose a suitable package source file.
    • The package will now be displayed in the Project Manager with a .bpl extension. Right click the package's entry and select Add to display the Add dialog box. Enter the full path to PJMenuSpeedButtons.pas in the Unit file name edit box (use the Browse button to navigate to the file). Click OK to add the component to the package.
    • Back in the Project Manager, check that all the added files appear under the package's Contains node. If so, right click the package's entry again and select Compile. If the resulting compilation succeeds a message should be displayed confirming that TPJMenuSpeedButton, TPJLinkedSpeedButton and TPJLinkedMenuSpeedButton have been registered. The components should now appear in the "DelphiDabbler" section of the Tool Palette. (Note that this will only appear when a form is selected in the designer).
    • If the components do not appear, the package may not have been installed. Install it from the Project Manager by selecting the package's Install right click menu option.
    • Close the package project using the File | Close menu item. Accept any prompt that offers to save the package project.
  4. If you wish you can now delete PJMenuSpeedButtons.pas and from the chosen installation folder.
  5. The supplied help file cannot be integrated with the IDE's help system. However it can be used independently of the IDE, albeit with reduced functionality.
  6. The demo program can be loaded into Delphi and compiled and run as required.

Installation with Delphi 4 to 7

The components must be installed into a design package. It is assumed that you will be installing into an existing package. You may wish to use the "user" package named dclusr.dpk or dclusrXX.dpk (where XX is a version number). This package can be found in $(DELPHI)\Lib. For Delphi 6 and 7 ensure that the chosen package "requires" designide.dcp.

  1. Decide upon a suitable location into which to install the components. This may be a subdirectory of your $(DELPHI)\lib folder.
  2. Copy PJMenuSpeedButtons.pas, PJMenuSpeedButtons.dcr and PJMenuSpeedButtons.res to the chosen installation folder.
  3. Start Delphi then:
    • Choose the Component | Install Component menu option to display the Install Component dialog box. Ensure the Into existing package tab is selected. Enter the full path to PJMenuSpeedButtons.pas in the Unit file name edit box. (Use the adjacent Browse button to navigate to the file). Choose the required package from the Package description combo box, or by navigating to the package using the Browse button. Click OK to confirm your selections.
    • A confirmation dialog box may displayed, querying whether it is OK to rebuild and reinstall the package. Click Yes to proceed.
    • The package will now be built and a dialog box should be displayed stating that the TPJMenuSpeedButton, TPJLinkedSpeedButton and TPJLinkedMenuSpeedButton components have been installed. Click OK to dismiss the dialog box.
    • The components will appear on the "DelphiDabbler" palette.
  4. If you wish you can now delete PJMenuSpeedButtons.pas from the chosen installation folder.
  5. The provided help file can be integrated with the IDE's OpenHelp system. Delphi 6 and 7 also require the use of the supplied .als file. The simplest way to install the help file is to use my Component Help Installer Expert (v2.2.1 or later). If you prefer to install manually, this article explains the process.
  6. The demo program can be loaded into Delphi and compiled and run as required.
